# WORM-MAN — The Underground Avenger

React + TypeScript fan-site for Worm-Man, built with Vite.
Spiritual successor to the Snake-Man HTML/CSS/JS project.

## Quick Start

```bash
# Install dependencies (requires Node 18+)
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Then open http://localhost:5173 in your browser.
